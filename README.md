## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:

3. Run the app:
   `npm run dev`



# AI Medical Assistant (Student Project)

## Project Overview
This project is an AI-powered medical assistant built using Google AI Studio and Gemini.
It provides non-emergency medical guidance, symptom explanations, and general health advice.

## Problem Statement
Many people struggle to access timely medical advice due to cost, location, and long wait times.
This platform helps users get basic medical guidance and understand symptoms before seeing a doctor.

## Features
- Health-related Q&A using AI
- Simple explanations of symptoms and illnesses
- Medical disclaimers and safety guardrails
- Non-emergency guidance only

## Tools & Technologies
- Google AI Studio (Gemini)
- HTML / TypeScript (generated by AI Studio)
- GitHub for version control
- Vercel / Netlify for deployment

## Disclaimer
This application is for educational purposes only and is not a replacement for professional medical advice.
